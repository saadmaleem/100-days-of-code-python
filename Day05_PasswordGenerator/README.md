# Day 05 — Password Generator 🔐

A Python program that generates a random secure password.

## What it does:
- Takes user input for number of letters, symbols and numbers
- Randomly generates a secure password
- Shuffles characters for extra security

## How to run:
1. Copy the code from `main.py`
2. Paste it into any Python editor:
   - [replit.com](https://replit.com)
   - [online-python.com](https://online-python.com)
   - VS Code / PyCharm
3. Run it ✅
